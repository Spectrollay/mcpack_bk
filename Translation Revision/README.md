<h3><a href="https://crowdin.com/project/translations-for-minecraft">Minecraft基岩版翻译(多语言) Crowdin</a></h3>

由俄罗斯玩家(同时也是Minecraft Java版的俄语校对员)开发,能将Java版的游戏翻译整合到基岩版,再由人工补足基岩版特有词条并制作成资源包.此项目疑似无限期弃坑.
